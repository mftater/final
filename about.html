About This Project

The Collaborative Calendar is a custom-built scheduling application designed to simplify how families, spouses, and roommates manage their shared responsibilities. Built entirely from scratch using HTML, CSS, JavaScript, and PHP, the project is a blend of minimal design and maximum usability.

Development Goals

  • Support for collaborative scheduling without overwriting others' data
  • Expandable backend using MySQL for storing users, events, and permissions
  • Responsive and accessible design
  • Security-minded features like addition limits and approval workflows
  • Simple user profiles and request-based calendar sharing
